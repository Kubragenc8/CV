# Kübra Genç Resume Website

This project is a personal resume website prepared using HTML and CSS.

## Project Purpose

The purpose of this project is to publish my resume on the web and present my education, technical skills, projects, certificates, and contact information in a clean and professional format.

## Technologies Used

- HTML
- CSS
- Font Awesome Icons
- Google Fonts

## Website Content

The resume website includes the following sections:

- About Me
- Skills
- Certificates
- Languages
- Projects and Experience
- Education
- Strengths
- Contact Icons

## Features

- Responsive design
- Modern resume layout
- Profile photo section
- Clickable contact icons
- GitHub, LinkedIn, email, and phone links
- Clean and simple user interface
